PRICE

Design System and TypeScript frontend for business-critical BI tools used daily by fund managers making investment decisions.

Role
Senior Software Engineer
Deliverables
Figma Design System · TypeScript Frontend
Users
Fund Managers · Internal Engineering
Context

Tools used to manage billions — built to a higher standard.

T. Rowe Price is one of the world’s largest active investment management firms. The BI tools I worked on were used daily by fund managers making decisions that move significant capital. A confusing interface or inconsistent data presentation isn’t a minor UX issue here — it’s a risk surface.

My role spanned both design and implementation: designing components in Figma, then building them in TypeScript. Every component needed to work perfectly across the full range of data densities, states, and edge cases a fund manager would encounter.

Design System

Building the foundation that all future work would stand on.

Alongside the BI tool work, I co-authored the strategy and initial component library for a new Design System serving the firm’s internal engineering teams. This is infrastructure work — invisible to end users, but the thing that determines whether every product built afterwards is consistent, accessible, and maintainable.

The design system work required thinking at two levels simultaneously: the individual component (does this button cover all its states?) and the system logic (how do these components combine? what are the rules? how do we prevent drift?). Getting both right is what separates a component library from a genuine design system.

In financial services, the design system isn’t a convenience. It’s how you ensure a fund manager never misreads a number because two components rendered inconsistently.
